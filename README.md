# reddiView (Reddit Client)

A feature-rich Reddit client built with **React**, **Redux**, and **Create React App (CRA)**. This application leverages the Reddit API to provide an engaging and user-friendly way to browse, interact with, and manage your Reddit account activities.

---

## Features

- **Browse Reddit Subreddits**:
  - View a curated list of popular subreddits.
  - Navigate subreddit content with a sleek, intuitive interface.
- **Upvote & Downvote Posts**:
  - Seamlessly interact with posts by upvoting or downvoting content, just like on Reddit.
- **Comment Interaction**:
  - Fetch and view comments for individual posts.
  - Skeleton loaders provide a smooth user experience during data fetches.
- **Subreddit Management**:
  - Browse through your subscribed subreddits and interact with their content.
  - Quickly switch between subreddits with ease.
- **Home Feed**:
  - Access a personalized feed of top posts across Reddit.
- **Responsive Design**:
  - Optimized for use across devices, including desktops, tablets, and smartphones.

---

## Tech Stack

- **React**: For building a dynamic and component-based user interface.
- **Redux**: State management for global application data, including subreddits, posts, and votes.
- **Redux Thunk**: Middleware for handling asynchronous actions.
- **Create React App (CRA)**: Boilerplate for a streamlined React development experience.
- **CSS Modules**: Scoped styles for better maintainability and modularity.
